# Akan-name-generator

## Project Description
This is a simple web application that calculates the day of the week a user was born and assigns them an Akan name based on their gender.<br>
Akan names come from Ghanaian culture, where children are named based on the day they were born.

## Author's Information
Mark Israel - https://github.com/israelatsenga-arch

## Setup Instructions
1. Clone the repository
2. Open the project
3. Enter your birthdate and gender
4. Submit your answers then receive your Akan name

## BDD(Behavior-Driven Development)
1. Input:Birthdate and gender<br>
    * Output:An Akan name based on your birthdate and gender
2. Input:Invalid date(day<=0 or day >31)<br>
    * Output:Incorrect enter a valid date
3. Input:Invalid month(month<=0 or month >12)<br>
    * Output:Incorrect enter a valid month
